About

This website was created as a result of a high school physics project. After doing many kinematic equation problems, we wanted a way to solve them more easily.

The goal of this website was to create a way to do kinematic physics problems. This was accomplished using HTML, CSS, and Javascript. The site is hosted on GitHub pages.

You can check out our GitHub to see our progress on this project, and feel free to contribute by forking the repository and submitting a pull request!
